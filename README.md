# SiteTrack Pro - Construction Intelligence Demo

A real-time construction site tracking and lead management demonstration.

## Features

- Real-time GPS tracking on OpenStreetMap
- Automated route following along actual streets
- Lead capture with detailed forms
- Photo upload simulation
- Playback controls (pause/resume/stop)
- Leads panel with organized view
- Fully responsive (mobile & desktop)

## Deploy to Vercel

```bash
# Install Vercel CLI globally
npm install -g vercel

# Deploy
vercel --prod
```

## Local Development

Simply open `index.html` in a modern web browser.

## Technologies

- Leaflet.js for mapping
- OpenStreetMap tiles via CARTO
- OSRM for routing
- Tailwind CSS for styling
- Lucide icons
