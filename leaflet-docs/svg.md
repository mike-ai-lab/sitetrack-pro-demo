SVG. Inherits`Renderer`.

Due totechnical limitations, SVG is not available in all web browsers, notably Android 2.x and 3.x.

Although SVG is not available on IE7 and IE8, these browsers supportVML(a now deprecated technology), and the SVG renderer will fall back to VML in this case.

VML was deprecated in 2012, which means VML functionality exists only for backwards compatibility with old versions of Internet Explorer.

Use SVG by default for all paths in the map:

```
var map = L.map('map', {
	renderer: L.svg()
});

```

Use a SVG renderer with extra padding for specific vector geometries:

```
var map = L.map('map');
var myRenderer = L.svg({ padding: 0.5 });
var line = L.polyline( coordinates, { renderer: myRenderer } );
var circle = L.circle( center, { renderer: myRenderer } );

```
